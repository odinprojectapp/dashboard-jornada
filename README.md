# Dashboard Jornada Laboral — GRC Sincelejo

Dashboard web para análisis de jornada laboral con semáforo SIC, métricas y filtros dinámicos.

**Paleta corporativa:** Gold `#d1a33c` · Grey `#c7c7c7` · Black `#000000`

## URL del dashboard
https://odinprojectapp.github.io/dashboard-jornada/

## Para actualizar datos
1. Edita `data.js` con los nuevos registros
2. Sube el archivo al repo
3. GitHub Pages actualiza en ~1 minuto
